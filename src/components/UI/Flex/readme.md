category: Components

title: Flex

---

```javascript
<Flex size={SIZE.small} prop1="value prop 1" prop2={50000} />
```

| Property | Description         | Type   | Default |
| -------- | ------------------- | ------ | ------- |
| size     | size component      | string | 'big'   |
| prop1    | prop 1 of component | string | 'value' |
| size     | prop 2 of component | number | 2000    |
